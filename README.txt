WEBSITE PEMERINTAH PROVINSI DIY
==============================================

Website resmi Pemerintah Provinsi Daerah Istimewa Yogyakarta

STRUKTUR FILE:
- index.html       : Halaman Beranda
- profil.html      : Halaman Profil (dengan foto Gubernur & Wakil Gubernur)
- peta.html        : Halaman Peta Yogyakarta (menggunakan peta yang Anda upload)
- potensi.html     : Halaman Potensi Daerah (dengan foto real)
- pariwisata.html  : Halaman Pariwisata (dengan foto real)
- kontak.html      : Halaman Kontak
- styles.css       : File CSS untuk semua halaman
- peta-yogyakarta.png : Gambar peta yang Anda upload

FITUR:
✅ Multi-page website (tidak single page)
✅ Peta Yogyakarta menggunakan gambar yang Anda upload
✅ Foto real untuk potensi daerah dan pariwisata (dari Unsplash)
✅ Profil lengkap dengan foto Gubernur dan Wakil Gubernur
✅ Desain profesional dan modern
✅ Responsive untuk desktop dan mobile
✅ Navigasi antar halaman yang smooth

CARA MENGGUNAKAN:
1. Extract semua file ke dalam satu folder
2. Buka file index.html di browser
3. Navigasi antar halaman menggunakan menu di header

PERUBAHAN DARI WEBSITE ASLI:
❌ Dihapus: PPID, Informasi & Berita, Layanan Publik
✨ Ditambahkan: Peta Yogyakarta, Potensi Daerah, Pariwisata, Profil dengan foto pemimpin

DESAIN:
- Warna tema: Cokelat & Emas (tradisional & elegan)
- Font: Playfair Display, Lora, Montserrat
- Gaya: Profesional, Modern, Berkelas
